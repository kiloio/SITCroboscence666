## 车辆接线示意图

```mermaid
graph TD

    Pi[RDK X3]

    subgraph 控制板
        电源管理
        MCU[MCU<br>STM32]
        IMU
        电机驱动A
        电机驱动B
    end

    电池 --> 电源管理
    控制板 -- 供电 --> Pi
    Pi <-- 串口通讯 --> MCU
    IMU <--> MCU
    MCU --> 电机驱动A
    MCU --> 电机驱动B
    电机A -- 编码 --> MCU
    电机B -- 编码 --> MCU
    电机驱动A -- 速度控制 --> 电机A
    电机驱动B -- 速度控制 --> 电机B
    Pi <--USB--> 雷达

```
