## 车辆接线示意图

```mermaid
graph TD

    Pi[RDK X3 Pi]

    subgraph 控制板
        电源管理
        STM32
        陀螺仪
        电机驱动A
        电机驱动B
    end

    电池 --> 电源管理
    控制板 -- 供电 --> Pi
    Pi <-- 串口通讯 --> STM32
    陀螺仪 <--> STM32
    STM32 --> 电机驱动A
    STM32 --> 电机驱动B
    电机A -- 编码 --> STM32
    电机B -- 编码 --> STM32
    电机驱动A -- 速度控制 --> 电机A
    电机驱动B -- 速度控制 --> 电机B

```
